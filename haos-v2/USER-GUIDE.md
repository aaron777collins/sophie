## ğŸš€ Getting Started

### Creating an Account

1. Visit the HAOS web application
2. Click "Create Account"
3. Choose a username
4. Select a Matrix homeserver
   - Default: matrix.org
   - Or use a custom homeserver
5. Set a strong password
6. Verify your email (optional but recommended)

### Logging In

- Enter your username or email
- Provide your password
- Optional: Use two-factor authentication
