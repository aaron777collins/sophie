## 2026-03-02 Session - Matrix Voice Chat Fixes

### [00:23 - 01:15 EST] Matrix Voice Chat Debugging with Aaron

**Problem:** Aaron's mic was being forced disabled in Element X when trying to voice chat.

**Root Causes Found:**
1. Traefik had no TLS certificates for port 5350 (TURNS)
   - The playbook configures `external_tls: true` expecting Traefik to terminate TLS
   - But Traefik had `certificatesResolvers: null` since Caddy handles TLS
   
2. TURN relay ports (30000-40000) were not exposed
   - LiveKit was in Docker bridge network
   - Only ports 7880-7882, 3479, 5350 were published
   - Relay ports needed for actual media flow

3. Caddy couldn't reach LiveKit after switching to host networking
   - Changed from container DNS name to Docker host gateway IP

**Fixes Applied:**
1. Copied Caddy TLS certs to LiveKit, set `external_tls: false`
2. Switched LiveKit to `--network=host` for relay port access
3. Updated Caddyfile to route via `172.18.0.1:7880`
4. Fixed file permissions (chmod 644) on configs

**Infrastructure Now:**
- TLS handled by LiveKit directly on port 5350
- All ports accessible via host networking
- Caddy routes WebSocket traffic via host gateway
